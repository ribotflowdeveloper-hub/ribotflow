# Finances: Pressupostos (Quotes)

La secció de Pressupostos et permet crear, enviar i gestionar les teves propostes comercials.

## Com crear un pressupost?
1.  Ves a "Finances" > "Pressupostos".
2.  Fes clic a "Crear Pressupost".
3.  Associa el pressupost a un contacte (del CRM).
4.  Afegeix línies de "Concepte". Aquests conceptes (productes o serveis) els pots seleccionar del teu catàleg.
5.  Especifica quantitat, preu i impostos per a cada línia.

## Estats d'un pressupost
Un pressupost pot tenir diversos estats:
* **Draft (Esborrany):** Encara l'estàs editant.
* **Sent (Enviat):** L'has enviat al client.
* **Accepted (Acceptat):** El client l'ha aprovat.
* **Declined (Rebutjat):** El client l'ha rebutjat.
* **Invoiced (Facturat):** Ja has generat una factura a partir d'aquest pressupost.

## Convertir un pressupost a factura
Quan un client accepta un pressupost (estat "Accepted"), pots convertir-lo directament en una factura. Ves al detall del pressupost i busca l'opció "Convertir a Factura". Això crearà automàticament una factura nova amb totes les dades del pressupost.
