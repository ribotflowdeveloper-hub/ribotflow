# Mòdul Network

El mòdul Network és una eina de visualització geogràfica. Et permet veure els teus contactes i altres usuaris de RibotFlow en un mapa.

Aquesta funció t'ajuda a identificar clients o col·laboradors en una àrea geogràfica específica, facilitant la planificació de reunions presencials o l'anàlisi de la teva presència al mercat.
