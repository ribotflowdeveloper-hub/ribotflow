# Finances: Despeses i Proveïdors

## Despeses (Expenses)
Aquesta secció t'ajuda a portar un registre de tots els costos i despeses del teu negoci (lloguer, material, programari, etc.).

Per registrar una despesa:
1.  Ves a "Finances" > "Despeses".
2.  Fes clic a "Nova Despesa".
3.  Introdueix l'import, la data i una descripció.
4.  Opcionalment, pots assignar la despesa a un "Proveïdor".
5.  Pots pujar un rebut o factura adjunta per portar un millor control.

## Proveïdors (Suppliers)
És una base de dades per gestionar les empreses o persones a qui compres serveis o productes. Pots vincular despeses a proveïdors per saber quant gastes amb cadascun.
