# CRM: Pipeline de Vendes (Oportunitats)

El Pipeline és una eina visual per gestionar el teu procés de vendes. Està organitzat en columnes que representen les etapes de la venda.

## Què és una Oportunitat?
Una oportunitat és un tracte o venda potencial. Cada oportunitat té un valor econòmic, un contacte associat i es troba en una etapa específica del pipeline.

## Les Etapes del Pipeline
Les etapes per defecte són:
* Nou Lead
* Contactat
* Proposta Enviada
* Negociació
* Guanyat
* Perdut

## Com funciona?
Pots arrossegar i deixar anar (drag-and-drop) les targetes d'oportunitat entre les columnes per actualitzar-ne l'estat. També pots fer clic a una oportunitat per editar-ne els detalls, afegir notes o crear-hi pressupostos directament.

## Com crear una oportunitat?
1.  Ves a "CRM" > "Pipeline".
2.  Fes clic a "Crear Oportunitat".
3.  Assigna-li un nom, un contacte (del teu llistat de contactes) i un valor estimat.
4.  S'afegirà a la primera columna.
